# Restaurant Review Sentiment Analysis

This project performs sentiment analysis on restaurant reviews using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes customer reviews, predicts sentiment (positive/negative), and compares restaurants based on customer feedback.

## Features
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Sentiment prediction using Logistic Regression
- Restaurant comparison based on review sentiment
- Custom collected restaurant review dataset

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## Project Workflow
1. Data Collection
2. Text Cleaning
3. Feature Extraction using TF-IDF
4. Model Training
5. Sentiment Prediction
6. Restaurant Comparison

## Dataset
The project uses:
- Kaggle Restaurant Reviews Dataset
- Custom collected reviews for Mehfil, Paradise, and Bawarchi restaurants

## Results
The model predicts review sentiment and compares restaurants based on positive feedback analysis.
