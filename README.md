# Restaurant Review Sentiment Analysis and Analytics Dashboard

This project performs sentiment analysis on restaurant reviews using Natural Language Processing (NLP), Machine Learning, Streamlit, and Power BI. The system analyzes customer reviews, predicts sentiment (positive/negative), compares restaurants based on customer feedback, and visualizes insights through interactive dashboards.

## Features

* Text preprocessing and cleaning
* TF-IDF feature extraction
* Sentiment prediction using Logistic Regression
* Restaurant comparison based on customer sentiment
* Analysis of reviews from Mehfil, Paradise, and Bawarchi
* Streamlit web application for real-time prediction
* Power BI dashboard for analytics visualization
* Business insights from customer reviews

8. Streamlit Web App Deployment
## Dataset

The project uses:

* Kaggle Restaurant Reviews Dataset
* Custom collected reviews for Mehfil, Paradise, and Bawarchi restaurants

## Dashboard & Visualization

The project includes a Power BI dashboard to visualize:

* Restaurant sentiment comparison
* Positive vs Negative sentiment analysis
* Customer review insights
* Restaurant performance analytics

## Results

The model predicts customer sentiment and compares restaurants based on positive review percentages.

Final comparison result:

* Bawarchi → 40% Positive Reviews
* Paradise → 40% Positive Reviews
* Mehfil → 30% Positive Reviews
